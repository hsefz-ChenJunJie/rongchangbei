# 荣昶杯项目

## 项目概述
基于 AI 的智能对话辅助项目，集成语音转文字 (STT)、文字转语音 (TTS) 和大语言模型 (LLM) 技术。

## 核心功能
1. **语音输入**：用户语音输入对话内容
2. **智能建议**：AI 根据上下文生成回答建议
3. **语音输出**：将回答转换为语音播放
4. **交互优化**：支持建议修改和重新生成
